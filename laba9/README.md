## 5. Поиск строк по шаблону `7[135]` с помощью `grep`

Создание файла `numbers.txt`:

```bash
cat > numbers.txt << 'EOF'
123
178
176
755
713
873
181
EOF
```

Поиск строк, содержащих цифру `7`, за которой следует одна из цифр `1`, `3` или `5`:

```bash
grep -E '7[135]' numbers.txt
```

Результат:

```text
755
713
873
```

## 6. Поиск строк, начинающихся на `s` и заканчивающихся на `r`

Создание файла `words.txt`:

```bash
cat > words.txt << 'EOF'
starfish
starless
samscripter
stellar
microsrar
ascender
sacrifice
scalar
EOF
```

Поиск строк по шаблону: начинается на `s`, заканчивается на `r`:

```bash
grep -E '^s.*r$' words.txt
```

Результат:

```text
samscripter
stellar
scalar
```

## 7. Поиск корректных простых e-mail адресов

Создание файла `emails.txt`:

```bash
cat > emails.txt << 'EOF'
user@example.com
user.name@site.org
bad@address
@nope.com
name@site
user+tag@mail-domain.ru
EOF
```

Поиск строк с простыми корректными адресами e-mail:

```bash
grep -E '^[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}$' emails.txt
```

Результат:

```text
user@example.com
user.name@site.org
user+tag@mail-domain.ru
```

## 8. Демонстрация работы `tr` и поиск корректных IPv4-адресов

### 8.1. Работа утилиты `tr` на примере

Создание файла:

```bash
echo "Hello World 123" > text.txt
```

Перевод строчных букв в заглавные:

```bash
cat text.txt | tr 'a-z' 'A-Z'
```

Результат:

```text
HELLO WORLD 123
```

Удаление цифр:

```bash
cat text.txt | tr -d '0-9'
```

Результат:

```text
Hello World
```

Замена пробелов на подчёркивания:

```bash
cat text.txt | tr ' ' '_'
```

Результат:

```text
Hello_World_123
```

### 8.2. Поиск корректных IPv4-адресов

Создание файла `ips.txt`:

```bash
cat > ips.txt << 'EOF'
127.0.0.1
255.255.255.255
12.34.56
123.256.0.0
1.23.099.255
0.79.378.111
8.8.8.8
192.168.0.1
EOF
```

Пример корректной регулярки для поиска допустимых четырёхбайтовых IPv4-адресов:

```bash
grep -E '^((25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])\.){3}(25[0-5]|2[0-4][0-9]|1[0-9]{2}|[1-9]?[0-9])$' ips.txt
127.0.0.1
255.255.255.255
8.8.8.8
192.168.0.1
```

## 9. Поиск корректных номеров телефонов ведомственной АТС (000–399)

Создание файла `phones.txt`:

```bash
cat > phones.txt << 'EOF'
000
001
010
123
399
400
900
0
777
045
EOF
```

По условию корректны номера от `000` до `399` включительно. Поиск таких номеров:

```bash
grep -E '^([0-2][0-9]{2}|3[0-9]{2})$' phones.txt
```

Результат:

```text
000
001
010
123
399
045
```
